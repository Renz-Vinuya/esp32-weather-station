\#ESP32 Temperature \& Humidity Checker



A simple IoT project using an ESP32 to read data from a DHT22 sensor and display it on an SSD1306 OLED screen.

\## Components

\- \*\*ESP32\*\* (Microcontroller)

\- \*\*DHT22\*\* (Temperature and Humidity Sensor)

\- \*\*SSD1306 0.96" OLED\*\* (Display)



\## Wiring

\- \*\*OLED SDA\*\* -> ESP32 GPIO 21

\- \*\*OLED SCL\*\* -> ESP32 GPIO 22

\- \*\*DHT22 Data\*\* -> ESP32 GPIO 15
